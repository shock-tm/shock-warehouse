# missingPluginDataDelete

shock tm

https://adodtp.com/

===

"missingPluginDataDelete" is a plugin that deletes plugin data that is not in the environment.

「missingPluginDataDelete」は環境にないプラグインデータを削除するプラグインです。

### Compatible apps
InDesign 2025 - 2026 (20 - 21)  
Windows only

### Install
Step 1:  
Download "missingPluginDataDelete [version] .zip" for the version you want to install.

インストールしたいバージョンの「missingPluginDataDelete [version] .zip」をダウンロードします。

Step 2:  
Copy the extracted "shockPlugin" folder to "C:\Program Files\Adobe\Adobe InDesign [verison]\Plug-Ins".

解凍した「shockPlugin」フォルダを「C:\Program Files\Adobe\Adobe InDesign [verison]\Plug-Ins」にコピーします。

Step 3:  
Launch InDesign.

InDesignを起動します。

### execution
From the "Plugins" menu, select "Delete plugin information that not" / "For all documents"

メニュー「プラグイン」から「環境にないプラグイン情報の削除」/「全ドキュメント対象」で実行します。

### Uninstall
Delete the "C:\Program Files\Adobe\Adobe InDesign [verison]\Plug-Ins\shockPlugin" folder.

「C:\Program Files\Adobe\Adobe InDesign [verison]\Plug-Ins\shockPlugin」フォルダを削除します。
